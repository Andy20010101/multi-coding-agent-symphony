# v13.1 Release Evidence: Workbench Chinese Presentation Layer

Date: 2026-05-25

## Scope

v13.1 localizes the visible `symphony console` Workbench presentation layer to Chinese.

- Static Workbench labels, navigation, headings, badges, buttons, empty states, and detail labels render in Chinese.
- Runtime status values, booleans, command labels, command descriptions, filter labels, timeline labels, and known risk titles/details are mapped to Chinese for display.
- JSON contract field names, persisted status enum values, raw/debug JSON blocks, and copy-only command strings remain unchanged.
- The static `symphony diagnose --html` report remains on its existing English contract surface.

- Risk heading rendered as `前三风险`.
- Known dynamic risk detail rendered as `写入了 Symphony 运行时产物。`

- Screenshot captured at `/tmp/symphony-v13-1-workbench-cn-final.png`.

## Residual Risks

- The localization is a presentation mapping inside the embedded static Workbench. Unknown future server-provided labels may display as their raw contract values until added to the mapping table.
- Raw/debug JSON is intentionally not translated so automation contracts remain inspectable.
